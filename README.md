# ABDOMEN-BT-TIBB-KARAR-DESTEK-S-STEM-
Abdomen BT görüntülerinde derin öğrenme tabanlı modellerle patoloji tespiti yapan; model performanslarını karşılaştıran, açıklanabilir yapay zekâ yöntemleriyle sonuçları görselleştiren ve tıbbi görüntü analizi süreçleri için karar destek prototipi sunan kapsamlı yapay zekâ projesidir.
